## Quick Start

### Backend
```bash
cd backend
npm install
cp .env.example .env
# Fill out MONGODB_URI, JWT_SECRET, etc.
npm start
```

### Frontend
```bash
cd frontend
npm install
npm start
```

Then open [http://localhost:3000](http://localhost:3000).
